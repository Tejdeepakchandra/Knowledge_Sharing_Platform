What is Python?

Python is a high-level, interpreted, general-purpose programming language created by Guido van Rossum and first released in 1991. It emphasizes code readability with its notable use of significant whitespace and supports multiple programming paradigms including procedural, object-oriented, and functional programming.

History of Python

Python was conceived in the late 1980s by Guido van Rossum at Centrum Wiskunde & Informatica (CWI) in the Netherlands as a successor to the ABC language. Python 2.0 was released in 2000, and Python 3.0 (a major, backwards-incompatible release) was released in 2008.

  • First released in February 1991 (version 0.9.0)
  • Python 2.0 released in 2000 introduced list comprehensions
  • Python 3.0 released in 2008 was a major revision not completely backwards-compatible
  • Python 2 was officially discontinued in 2020

Key Features of Python

  • Easy to Learn: Python has a simple syntax similar to the English language.
  • Interpreted Language: Python is processed at runtime by the interpreter.
  • Cross-platform: Python runs on Windows, Linux, Mac, etc.
  • Large Standard Library: Comes with batteries included for many tasks.
  • Dynamic Typing: No need to declare variable types.
  • Memory Management: Automatic garbage collection.
  • Community Support: Large active community with many third-party packages.

Python Code Execution

  1. Writing the Program: Developers write Python code in a text editor or IDE.
  2. Parsing: The Python interpreter parses the source code into bytecode.
  3. Compilation: The bytecode is compiled into platform-independent Python bytecode (.pyc files).
  4. Execution: The Python Virtual Machine (PVM) executes the bytecode.

Conclusion

Python is a versatile and powerful programming language that is easy to learn yet capable of handling complex tasks. Its simplicity and readability make it an excellent choice for beginners and professionals alike.

Python Installation & Setup

Follow these steps to install Python on different operating systems and set up your development environment.

Installing Python

Python comes pre-installed on most Unix systems including Linux and macOS, but you may want to install a newer version.

Installing on Windows

  1. Download the latest Python installer from python.org
  2. Run the installer and check "Add Python to PATH"
  3. Click "Install Now" and follow the prompts
  4. Verify installation by opening Command Prompt and typing: 
    python --version

Installing on Linux

  1. Most Linux distributions come with Python pre-installed. Check version: 
    python3 --version
  2. To install Python 3 on Debian/Ubuntu: 
    sudo apt update
sudo apt install python3

Installing on macOS

  1. macOS comes with Python 2.7 pre-installed. For Python 3: 
    brew install python
    or download from python.org
  2. Verify installation: 
    python3 --version

Python Syntax Basics

Python has a clean and readable syntax that emphasizes code readability.

Structure of a Python Program

A basic Python program consists of statements that are executed sequentially:

# This is a comment
print("Hello, Python!")  # This prints a message

x = 5  # Variable assignment
y = 10
result = x + y

print(f"The result is {result}")  # f-string for string interpolation

Key Components

  • Statements: Instructions that Python can execute
  • Variables: Names that refer to values
  • Expressions: Combinations of values, variables, and operators
  • Comments: Explanatory notes ignored by the interpreter (# for single-line)
  • Indentation: Used to define code blocks (instead of curly braces)

Basic Syntax Rules

  • Python is case-sensitive
  • Statements don't need to end with semicolons (but can)
  • Code blocks are defined by indentation (typically 4 spaces)
  • Variables don't need type declarations
  • Function calls use parentheses ()

Running Python Code

Python code can be executed in several ways:

# From command line
python script.py

# Interactive interpreter
python
>>> print("Hello")
Hello

Python also supports interactive notebooks (Jupyter) for exploratory coding.

Python Variables & Data Types

Variables in Python are used to store data values. Python has various built-in data types.

Declaring Variables

Variables are created when you assign a value to them:

x = 5           # Integer
y = 3.14        # Float
name = "Alice"  # String
is_active = True # Boolean

Basic Data Types

  • int: Integer numbers (3, -5, 100)
  • float: Floating point numbers (3.14, -0.001)
  • str: Strings of characters ("hello", 'Python')
  • bool: Boolean values (True, False)
  • NoneType: Special type with one value None

Type Conversion

Python provides functions to convert between types:

num = "123"
num_int = int(num)  # Convert to integer

pi = 3.14
pi_str = str(pi)    # Convert to string

flag = 1
flag_bool = bool(flag) # Convert to boolean

Dynamic Typing

Python is dynamically typed - variable types are determined at runtime:

x = 5      # x is an integer
x = "hello" # Now x is a string

Variable Naming Rules

  • Can contain letters, digits, and underscores
  • Must begin with a letter or underscore
  • Cannot be a Python keyword
  • Case-sensitive
  • Should be descriptive (follow PEP 8 guidelines)

Conclusion

Python's flexible variable system and rich set of data types make it easy to work with different kinds of data.

Python Operators

Operators in Python are used to perform operations on variables and values.

Types of Operators

  • Arithmetic: +, -, *, /, // (floor division), % (modulus), ** (exponent)
  • Comparison: ==, !=, >, <, >=, <=
  • Logical: and, or, not
  • Assignment: =, +=, -=, *=, /=, etc.
  • Identity: is, is not (compare memory locations)
  • Membership: in, not in (check if value exists in sequence)
  • Bitwise: &, |, ^ (XOR), ~ (NOT), <<, >>

Example Program

# Arithmetic operators
print(10 + 3)   # 13
print(10 - 3)   # 7
print(10 * 3)   # 30
print(10 / 3)   # 3.333...
print(10 // 3)  # 3 (floor division)
print(10 % 3)   # 1 (modulus)
print(10 ** 3)  # 1000 (exponent)

# Comparison operators
print(10 > 3)   # True
print(10 == 10) # True
print(10 != 5)  # True

# Logical operators
print(True and False)  # False
print(True or False)   # True
print(not True)        # False

# Assignment operators
x = 5
x += 3  # Equivalent to x = x + 3
print(x) # 8

# Identity operators
a = [1, 2, 3]
b = a
print(a is b)  # True (same object)

Operator Precedence

Operators are evaluated in a specific order when they appear together:

  1. Parentheses ()
  2. Exponentiation **
  3. Multiplication/division/modulus/floor division (*, /, %, //)
  4. Addition/subtraction (+, -)
  5. Comparison operators (<, <=, >, >=, ==, !=)
  6. Identity operators (is, is not)
  7. Membership operators (in, not in)
  8. Logical NOT (not)
  9. Logical AND (and)
  10. Logical OR (or)

Conclusion

Operators are essential for performing computations and making decisions in Python programs.

Python Control Statements

Control statements determine the flow of execution in a Python program.

Types of Control Statements

  • Conditional: if, elif, else
  • Loops: for, while
  • Loop Control: break, continue, pass

Example Program

# if-elif-else
age = 20
if age < 13:
    print("Child")
elif age < 20:
    print("Teenager")
else:
    print("Adult")

# for loop
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# while loop
count = 0
while count < 5:
    print(count)
    count += 1

# break and continue
for num in range(10):
    if num == 3:
        continue  # Skip this iteration
    if num == 8:
        break     # Exit the loop
    print(num)

Match-Case (Python 3.10+)

Python 3.10 introduced pattern matching similar to switch-case:

status = 404
match status:
    case 200:
        print("OK")
    case 404:
        print("Not Found")
    case _:  # Default case
        print("Unknown status")

Conclusion

Control statements allow you to create complex program logic and handle different execution paths in Python.

Python Lists

Lists in Python are ordered, mutable collections that can hold items of different types.

Creating Lists

Lists are created using square brackets:

fruits = ["apple", "banana", "cherry"]  # List of strings
numbers = [1, 2, 3, 4, 5]              # List of integers
mixed = [1, "hello", 3.14, True]       # Mixed types

Accessing List Elements

List elements are accessed using their index (starting from 0):

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # "apple"
print(fruits[-1]) # "cherry" (negative index counts from end)

List Operations

Lists support various operations:

numbers = [1, 2, 3]

# Modifying elements
numbers[1] = 20  # [1, 20, 3]

# Adding elements
numbers.append(4)      # [1, 20, 3, 4]
numbers.insert(1, 1.5) # [1, 1.5, 20, 3, 4]

# Removing elements
numbers.remove(20)     # [1, 1.5, 3, 4]
popped = numbers.pop() # [1, 1.5, 3], returns 4

# Slicing
print(numbers[1:3])    # [1.5, 3]

List Methods

  • append(): Adds an element at the end
  • extend(): Adds elements from another list
  • insert(): Inserts an element at a specific position
  • remove(): Removes the first matching element
  • pop(): Removes and returns an element at given index
  • sort(): Sorts the list
  • reverse(): Reverses the list
  • copy(): Returns a shallow copy

Conclusion

Lists are versatile and fundamental data structures in Python that can store collections of items efficiently.

Python Tuples

Tuples are ordered, immutable collections that can hold items of different types.

Creating Tuples

Tuples are created using parentheses:

coordinates = (10, 20)          # Tuple of integers
colors = ("red", "green", "blue") # Tuple of strings
single = (5,)                   # Single-element tuple (note comma)

Accessing Tuple Elements

Tuple elements are accessed using their index (starting from 0):

colors = ("red", "green", "blue")
print(colors[0])  # "red"
print(colors[-1]) # "blue" (negative index counts from end)

Tuple Operations

Tuples support various operations (but are immutable):

t1 = (1, 2, 3)
t2 = (4, 5, 6)

# Concatenation
t3 = t1 + t2  # (1, 2, 3, 4, 5, 6)

# Repetition
t4 = t1 * 2   # (1, 2, 3, 1, 2, 3)

# Membership
print(2 in t1) # True

# Slicing
print(t3[2:5]) # (3, 4, 5)

Tuple Methods

Tuples have only two methods:

  • count(): Returns number of occurrences of a value
  • index(): Returns first index of a value

When to Use Tuples

  • When you need an immutable sequence
  • When you need to use a collection as a dictionary key
  • When you need to ensure data integrity
  • When working with functions that return multiple values

Conclusion

Tuples are useful when you need an immutable collection of items, often used for fixed data.

Python Dictionaries

Dictionaries are unordered, mutable collections of key-value pairs.

Creating Dictionaries

Dictionaries are created using curly braces:

person = {"name": "Alice", "age": 25, "city": "New York"}  # Key-value pairs
empty_dict = {}                                           # Empty dictionary

Accessing Dictionary Elements

Dictionary elements are accessed using their keys:

person = {"name": "Alice", "age": 25}

print(person["name"])  # "Alice"
print(person.get("age")) # 25
print(person.get("city", "Unknown")) # "Unknown" (default if key doesn't exist)

Modifying Dictionaries

You can add, update, or remove key-value pairs:

person = {"name": "Alice", "age": 25}

# Adding a new key-value pair
person["city"] = "New York"  # {"name": "Alice", "age": 25, "city": "New York"}

# Updating an existing key
person["age"] = 26  # {"name": "Alice", "age": 26, "city": "New York"}

# Removing a key-value pair
del person["city"]  # {"name": "Alice", "age": 26}

Dictionary Methods

Dictionaries have several useful methods:

  • keys(): Returns a view object of all keys
  • values(): Returns a view object of all values
  • items(): Returns a view object of all key-value pairs
  • clear(): Removes all items
  • copy(): Returns a shallow copy of the dictionary

Iterating Through a Dictionary

You can iterate through keys, values, or key-value pairs:

person = {"name": "Alice", "age": 25}

# Iterating through keys
for key in person:
    print(key)

# Iterating through values
for value in person.values():
    print(value)

# Iterating through key-value pairs
for key, value in person.items():
    print(f"{key}: {value}")

Conclusion

Dictionaries are powerful data structures that allow you to store and manage data in key-value pairs, making data retrieval efficient.

Python Functions

Functions are reusable blocks of code that perform a specific task.

Defining a Function

def greet(name):
    return f"Hello, {name}!"

Calling a Function

print(greet("Alice"))  # Output: Hello, Alice!

Function Parameters

Functions can take parameters to operate on:

def add(a, b):
    return a + b

print(add(5, 3))  # Output: 8

Default Parameters

def greet(name="Guest"):
    return f"Hello, {name}!"

print(greet())        # Output: Hello, Guest!
print(greet("Alice")) # Output: Hello, Alice!

Return Statement

The return statement is used to send back a value from a function:

def square(x):
    return x * x

result = square(4)
print(result)  # Output: 16

Lambda Functions

Lambda functions are anonymous functions defined with the lambda keyword:

square = lambda x: x * x
print(square(5))  # Output: 25

Conclusion

Functions are a key feature in Python that promote code reusability and organization. Understanding how to define and use functions is essential for effective programming.

Python Classes & Objects

Python is an object-oriented programming language, which means it allows you to define classes and create objects.

Defining a Class

class Dog:
    def _init_(self, name, age):
        self.name = name
        self.age = age

Creating an Object

my_dog = Dog("Buddy", 3)

Accessing Attributes

print(my_dog.name)  # Output: Buddy
print(my_dog.age)   # Output: 3

Defining Methods

class Dog:
    def bark(self):
        return "Woof!"

Using Methods

print(my_dog.bark())  # Output: Woof!

Inheritance

class Puppy(Dog):
    def play(self):
        return "Playing!"

Conclusion

Classes and objects are fundamental to object-oriented programming in Python, allowing for encapsulation and code reuse.

Python Modules

Modules are files containing Python code that can define functions, classes, and variables.

Creating a Module

To create a module, simply save a Python file with a .py extension:

# my_module.py
def greet(name):
    return f"Hello, {name}!"

Importing a Module

import my_module

print(my_module.greet("Alice"))  # Output: Hello, Alice!

From Import

from my_module import greet

print(greet("Bob"))  # Output: Hello, Bob!

Conclusion

Modules help organize code into manageable sections, promoting code reuse and better structure.

File Handling in Python

Python provides built-in functions to create, read, update, and delete files.

Opening a File

file = open("example.txt", "r")  # Open for reading

Reading from a File

content = file.read()  # Read entire file

Writing to a File

Conclusion

Exception Handling in Python

Exceptions are errors that occur during program execution. Python provides a way to handle these errors gracefully.

Try and Except

try:
    result = 10 / 0  # This will raise an exception
except ZeroDivisionError:
    print("Cannot divide by zero!")

Finally Block

try:
    file = open("example.txt", "r")
except FileNotFoundError:
    print("File not found!")
finally:
    print("Execution completed.")

Conclusion

Exception handling is crucial for building robust applications that can handle unexpected errors gracefully.

Python Generators

Generators are a way to create iterators using a function that yields values instead of returning them.

Creating a Generator

def count_up_to(n):
    count = 1
    while count <= n:
        yield count
        count += 1

Using a Generator

for number in count_up_to(5):
    print(number)  # Outputs: 1 2 3 4 5

Conclusion

Generators provide a memory-efficient way to handle large datasets by yielding items one at a time.

Python Decorators

Decorators are a way to modify or enhance functions or methods without changing their code.

Creating a Decorator

def my_decorator(func):
    def wrapper():
        print("Something is happening before the function is called.")
        func()
        print("Something is happening after the function is called.")
    return wrapper

Using a Decorator

@my_decorator
def say_hello():
    print("Hello!")

say_hello()  # Outputs: Something is happening before... Hello! ...after the function is called.

Conclusion

Decorators are a powerful tool for enhancing the functionality of functions and methods in a clean and readable way.

Python Lambda Functions

Lambda functions are small anonymous functions defined with the lambda keyword.

Creating a Lambda Function

add = lambda x, y: x + y

Using a Lambda Function

print(add(5, 3))  # Output: 8

Conclusion

Lambda functions are useful for creating small, throwaway functions without the need for a full function definition.

Python List Comprehensions

List comprehensions provide a concise way to create lists based on existing lists.

Creating a List Comprehension

squares = [x**2 for x in range(10)]

Using a List Comprehension

print(squares)  # Output: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

Conclusion

List comprehensions are a powerful feature that allows for more readable and efficient list creation.
