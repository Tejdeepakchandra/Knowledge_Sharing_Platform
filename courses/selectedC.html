What is C?

C is a general-purpose, procedural programming language developed in 1972 by Dennis Ritchie at Bell Labs. It is one of the most widely used programming languages of all time and provides low-level access to memory and simple set of keywords.

History of C

C was developed by Dennis Ritchie at Bell Labs in 1972 as a successor to the B language. It was created to write the UNIX operating system and has since become the foundation for many other languages like C++, Java, and Python.

  • Originally developed for implementing the UNIX operating system
  • First standardized by ANSI in 1989 (C89)
  • Later standards include C99 (1999) and C11 (2011)
  • Influenced almost all modern programming languages

Key Features of C

  • Procedural Language: C follows a step-by-step approach to break down a task into functions.
  • Fast and Efficient: C is compiled to machine code which executes very quickly.
  • Portable: C programs can run on different hardware with minimal changes.
  • Low-level Access: Provides direct access to memory via pointers.
  • Structured Language: Supports functions, loops, conditionals for structured programming.
  • Rich Library: Comes with extensive standard library functions.
  • Extensible: New features can be added via libraries.

How C Code Executes?

  1. Writing the Program: Developers write C code in a text editor or IDE.
  2. Preprocessing: The preprocessor handles directives like #include and #define.
  3. Compiling: The compiler converts source code into assembly code.
  4. Assembling: The assembler converts assembly code to machine code.
  5. Linking: The linker combines object files and libraries into an executable.
  6. Execution: The operating system loads and runs the executable.

Conclusion

C is a powerful and efficient programming language that forms the foundation of modern computing. Its simplicity, speed, and low-level capabilities make it ideal for system programming, embedded systems, and performance-critical applications.

C Installation & Setup

Follow these steps to install a C compiler on different operating systems and set up your development environment.

Installing C Compiler

To compile and run C programs, you need a C compiler. The most common ones are GCC (GNU Compiler Collection) for Linux/macOS and MinGW for Windows.

Installing on Windows

  1. Download MinGW (Minimalist GNU for Windows) from mingw.org
  2. Run the installer and select "mingw32-base" and "mingw32-gcc-g++" packages
  3. Add MinGW's bin directory to your system PATH:
    • Right-click on "This PC" → Properties → Advanced system settings
    • Click "Environment Variables" → Under "System variables", find "Path" and click "Edit"
    • Add the path to MinGW's bin directory (e.g., C:\MinGW\bin)
  4. Verify installation by opening Command Prompt and typing: 
    gcc --version

Installing on Linux

  1. Open terminal and install GCC: 
    sudo apt update
sudo apt install build-essential
  2. Verify installation: 
    gcc --version

Installing on macOS

  1. Install Xcode Command Line Tools: 
    xcode-select --install
  2. Verify installation: 
    gcc --version

C Syntax Basics

C is a structured programming language with a relatively small set of keywords and a straightforward syntax.

Structure of a C Program

A basic C program consists of several components:

// Preprocessor directive
#include <stdio.h>

// Main function - program entry point
int main() {
    // Statement
    printf("Hello, C Programming!\n");
    
    // Return statement
    return 0;
}

Key Components

  • Preprocessor Directives: Lines starting with # that are processed before compilation (e.g., #include)
  • Main Function: Every C program must have a main() function where execution begins
  • Statements: Instructions that end with a semicolon (;)
  • Comments: Explanatory notes ignored by the compiler (// for single-line, /* */ for multi-line)

Basic Syntax Rules

  • C is case-sensitive
  • Statements end with a semicolon (;)
  • Code blocks are enclosed in curly braces {}
  • Whitespace is ignored (except in strings and character constants)
  • Function calls use parentheses ()

Compiling and Running

gcc program.c -o program
./program

This compiles program.c into an executable named 'program' and then runs it.

C Variables & Data Types

Variables in C are containers for storing data values, each with a specific type that determines size and layout.

Declaring Variables

Variable declaration specifies the type and optionally initializes the value:

int age = 25;          // Integer
float price = 19.99;    // Floating point
char grade = 'A';      // Character
double pi = 3.14159;  // Double precision

Basic Data Types

  • int: Integer numbers (typically 4 bytes)
  • float: Single-precision floating point (4 bytes)
  • double: Double-precision floating point (8 bytes)
  • char: Single character (1 byte)
  • void: No type (used for functions that don't return values)

Type Modifiers

Can alter the meaning of basic types:

  • signed: Can be positive or negative (default)
  • unsigned: Only positive values
  • short: Smaller storage size
  • long: Larger storage size

Example Program

#include <stdio.h>

int main() {
    int age = 30;
    float height = 5.9;
    char initial = 'J';
    
    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);
    printf("Initial: %c\n", initial);
    
    return 0;
}

Variable Naming Rules

  • Can contain letters, digits, and underscores
  • Must begin with a letter or underscore
  • Cannot be a C keyword
  • Case-sensitive

Conclusion

Understanding variables and data types is fundamental to C programming, as they define how data is stored and manipulated.

C Operators

Operators in C are symbols that perform operations on variables and values.

Types of Operators

  • Arithmetic: +, -, *, /, % (modulus)
  • Relational: ==, !=, >, <, >=, <=
  • Logical: && (AND), || (OR), ! (NOT)
  • Assignment: =, +=, -=, *=, /=, %=
  • Bitwise: &, |, ^ (XOR), ~ (NOT), <<, >>
  • Increment/Decrement: ++, --
  • Ternary: ? : (conditional operator)

Example Program

#include <stdio.h>

int main() {
    int a = 10, b = 5;
    
    // Arithmetic
    printf("Sum: %d\n", a + b);
    printf("Modulus: %d\n", a % b);
    
    // Relational
    printf("Is a > b? %d\n", a > b);
    
    // Logical
    int x = 1, y = 0;
    printf("x && y: %d\n", x && y);
    
    // Assignment
    a += 3; // a = a + 3
    printf("Updated a: %d\n", a);
    
    // Increment
    a++;
    printf("Incremented a: %d\n", a);
    
    return 0;
}

Operator Precedence

Operators are evaluated in a specific order when they appear together:

  1. Parentheses ()
  2. Unary operators (++, --, !, ~, +, -)
  3. Multiplication/division/modulus (*, /, %)
  4. Addition/subtraction (+, -)
  5. Relational operators (<, <=, >, >=)
  6. Equality operators (==, !=)
  7. Logical AND (&&)
  8. Logical OR (||)
  9. Assignment operators (=, +=, -=, etc.)

Conclusion

Operators are essential for performing computations and making decisions in C programs.

C Control Statements

Control statements determine the flow of execution in a C program.

Types of Control Statements

  • Decision Making: if, if-else, switch-case
  • Loops: for, while, do-while
  • Branching: break, continue, goto, return

Example Program

#include <stdio.h>

int main() {
    int number = 10;
    
    // if-else
    if (number > 0) {
        printf("%d is positive\n", number);
    } else if (number < 0) {
        printf("%d is negative\n", number);
    } else {
        printf("Number is zero\n");
    }
    
    // switch-case
    switch(number) {
        case 5:
            printf("Five\n");
            break;
        case 10:
            printf("Ten\n");
            break;
        default:
            printf("Other number\n");
    }
    
    // for loop
    printf("Counting from 1 to 5:\n");
    for (int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    printf("\n");
    
    // while loop
    int j = 5;
    while (j > 0) {
        printf("%d ", j);
        j--;
    }
    printf("\n");
    
    return 0;
}

Nested Control Structures

Control structures can be nested within each other:

#include <stdio.h>

int main() {
    for (int i = 1; i <= 3; i++) {
        printf("Outer loop iteration %d\n", i);
        
        for (int j = 1; j <= 2; j++) {
            printf("  Inner loop iteration %d\n", j);
            
            if (i == j) {
                printf("    i equals j\n");
            }
        }
    }
    return 0;
}

Conclusion

Control statements allow you to create complex program logic and handle different execution paths.

C Arrays

Arrays in C are collections of variables of the same type stored in contiguous memory locations.

Declaring Arrays

Syntax for array declaration and initialization:

int numbers[5]; // Declaration
int primes[] = {2, 3, 5, 7, 11}; // Declaration with initialization

Accessing Array Elements

Array elements are accessed using their index (starting from 0):

#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    
    printf("First element: %d\n", arr[0]);
    printf("Third element: %d\n", arr[2]);
    
    // Modifying an element
    arr[1] = 25;
    printf("Modified second element: %d\n", arr[1]);
    
    return 0;
}

Multi-dimensional Arrays

C supports arrays of arrays (2D, 3D, etc.):

#include <stdio.h>

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };
    
    printf("Element at [1][2]: %d\n", matrix[1][2]);
    
    // Nested loops to process 2D array
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\n");
    }
    
    return 0;
}

Array Limitations

  • Fixed size (must know size at compile time)
  • No bounds checking (can lead to buffer overflows)
  • Cannot be resized after declaration

Conclusion

Arrays are fundamental for storing and processing collections of data in C.

C Strings

In C, strings are arrays of characters terminated by a null character ('\0').

String Declaration

Multiple ways to declare strings:

#include <stdio.h>

int main() {
    char str1[] = "Hello"; // Automatically adds null terminator
    char str2[6] = {'H', 'e', 'l', 'l', 'o', '\0'};
    char str3[10]; // Can hold up to 9 characters + null terminator
    
    printf("str1: %s\n", str1);
    printf("str2: %s\n", str2);
    
    return 0;
}

String Functions

The string.h library provides useful functions:

#include <stdio.h>
#include <string.h>

int main() {
    char str1[20] = "Hello";
    char str2[] = "World";
    
    // String length
    printf("Length of str1: %lu\n", strlen(str1));
    
    // String concatenation
    strcat(str1, " ");
    strcat(str1, str2);
    printf("Concatenated: %s\n", str1);
    
    // String copy
    strcpy(str1, "Goodbye");
    printf("Copied: %s\n", str1);
    
    // String comparison
    printf("Comparison: %d\n", strcmp("apple", "banana"));
    
    return 0;
}

Common String Functions

  • strlen(): Returns length of string
  • strcpy(): Copies one string to another
  • strcat(): Concatenates two strings
  • strcmp(): Compares two strings
  • strstr(): Finds substring in string

Conclusion

Strings are essential for text processing in C, though they require careful handling due to their array-based nature.

C Functions

Functions in C are blocks of code that perform specific tasks and can be reused.

Function Definition

Basic structure of a function:

// Function declaration (prototype)
int add(int a, int b);

// Function definition
int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    printf("Sum: %d\n", result);
    return 0;
}

Function Components

  • Return Type: Data type of the value returned (void if none)
  • Function Name: Identifier used to call the function
  • Parameters: Variables that receive values passed to the function
  • Function Body: Block of code that defines what the function does

Function Types

  • Standard Functions: Built-in functions like printf(), scanf(), etc.
  • User-defined Functions: Functions created by the programmer to perform specific tasks

Example Program

#include <stdio.h>

// Function to calculate the square of a number
int square(int num) {
    return num * num;
}

int main() {
    int number = 4;
    printf("Square of %d is %d\n", number, square(number));
    return 0;
}

Function Overloading

C does not support function overloading (same function name with different parameters) like C++ does. Each function must have a unique name.

Conclusion

Functions are essential for organizing code, improving readability, and reusability in C programming.

C Pointers

Pointers are variables that store the memory address of another variable, allowing for efficient data manipulation.

Declaring Pointers

Syntax for declaring a pointer:

int *ptr; // Pointer to an integer

Using Pointers

To assign a pointer, use the address-of operator (&):

#include <stdio.h>

int main() {
    int var = 20;
    int *ptr = &var; // ptr now holds the address of var
    
    printf("Value of var: %d\n", var);
    printf("Value via pointer: %d\n", *ptr); // Dereferencing ptr
    
    return 0;
}

Pointer Arithmetic

Pointers can be incremented or decremented to point to the next or previous memory location:

#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30};
    int *ptr = arr; // Points to the first element
    
    printf("First element: %d\n", *ptr);
    ptr++; // Move to the next element
    printf("Second element: %d\n", *ptr);
    
    return 0;
}

Pointers and Arrays

Arrays and pointers are closely related; the name of an array acts as a pointer to its first element:

#include <stdio.h>

int main() {
    int arr[] = {1, 2, 3};
    int *ptr = arr; // Equivalent to int *ptr = &arr[0];
    
    for (int i = 0; i < 3; i++) {
        printf("Element %d: %d\n", i, *(ptr + i)); // Accessing elements using pointer arithmetic
    }
    
    return 0;
}

Conclusion

Pointers are a powerful feature in C that enable dynamic memory management and efficient data manipulation.

C Structures

Structures are user-defined data types that group related variables of different types under a single name.

Defining Structures

Syntax for defining a structure:

struct Person {
    char name[50];
    int age;
};

Declaring Structure Variables

You can declare variables of a structure type:

struct Person person1, person2;

Accessing Structure Members

Use the dot operator (.) to access members of a structure:

#include <stdio.h>

struct Person {
    char name[50];
    int age;
};

int main() {
    struct Person person1;
    
    // Assigning values
    strcpy(person1.name, "Alice");
    person1.age = 30;
    
    // Accessing values
    printf("Name: %s, Age: %d\n", person1.name, person1.age);
    
    return 0;
}

Structures and Functions

Structures can be passed to functions as parameters:

#include <stdio.h>

struct Person {
    char name[50];
    int age;
};

void printPerson(struct Person p) {
    printf("Name: %s, Age: %d\n", p.name, p.age);
}

int main() {
    struct Person person1 = {"Bob", 25};
    printPerson(person1);
    
    return 0;
}

Conclusion

Structures are useful for organizing complex data and are widely used in C programming.

C Unions

Unions are similar to structures but allow storing different data types in the same memory location.

Defining Unions

Syntax for defining a union:

union Data {
    int intValue;
    float floatValue;
    char charValue;
};

Declaring Union Variables

You can declare variables of a union type:

union Data data1, data2;

Accessing Union Members

Use the dot operator (.) to access members of a union:

#include <stdio.h>

union Data {
    int intValue;
    float floatValue;
    char charValue;
};

int main() {
    union Data data1;
    
    data1.intValue = 10;
    printf("Integer: %d\n", data1.intValue);
    
    data1.floatValue = 220.5; // Overwrites intValue
    printf("Float: %.2f\n", data1.floatValue);
    
    return 0;
}

Union Size

The size of a union is determined by the size of its largest member:

#include <stdio.h>

union Data {
    int intValue;
    float floatValue;
    char charValue;
};

int main() {
    printf("Size of union: %lu\n", sizeof(union Data));
    return 0;
}

Conclusion

Unions are useful for memory-efficient data storage when only one of the members is needed at a time.

C File Handling

File handling in C allows you to read from and write to files, enabling data persistence.

File Operations

  • Opening a File: Use fopen() to open a file.
  • Reading/Writing: Use functions like fscanf(), fprintf(), fread(), fwrite().
  • Closing a File: Use fclose() to close a file.

Example Program

#include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "w"); // Open for writing
    
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    fprintf(file, "Hello, File Handling in C!\n");
    fclose(file); // Close the file
    
    return 0;
}

Reading from a File

#include <stdio.h>

int main() {
    FILE *file;
    char buffer[100];
    
    file = fopen("example.txt", "r"); // Open for reading
    
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    while (fgets(buffer, sizeof(buffer), file) != NULL) {
        printf("%s", buffer);
    }
    
    fclose(file); // Close the file
    
    return 0;
}

Conclusion

File handling is essential for data storage and retrieval in C programming.

C Dynamic Memory

Dynamic memory allocation allows you to allocate memory at runtime using functions like malloc(), calloc(), realloc(), and free().

Dynamic Memory Functions

  • malloc(): Allocates a specified number of bytes and returns a pointer to the allocated memory.
  • calloc(): Allocates memory for an array of elements and initializes them to zero.
  • realloc(): Resizes previously allocated memory.
  • free(): Deallocates previously allocated memory.

Example Program

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *arr;
    int n = 5;
    
    arr = (int *)malloc(n * sizeof(int)); // Allocate memory for 5 integers
    
    if (arr == NULL) {
        printf("Memory allocation failed!\n");
        return 1;
    }
    
    for (int i = 0; i < n; i++) {
        arr[i] = i + 1; // Assign values
    }
    
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]); // Print values
    }
    
    free(arr); // Deallocate memory
    
    return 0;
}

Conclusion

Dynamic memory management is crucial for efficient memory usage in C programming.

C Recursion

Recursion is a programming technique where a function calls itself to solve a problem.

How Recursion Works

A recursive function must have a base case to terminate the recursion and avoid infinite loops.

Example Program

#include <stdio.h>

int factorial(int n) {
    if (n == 0) return 1; // Base case
    return n * factorial(n - 1); // Recursive call
}

int main() {
    int num = 5;
    printf("Factorial of %d is %d\n", num, factorial(num));
    return 0;
}

Advantages and Disadvantages

  • Advantages: Simplifies code for problems that can be broken down into smaller subproblems.
  • Disadvantages: Can lead to high memory usage and stack overflow if not managed properly.

Conclusion

Recursion is a powerful technique in C programming, but it requires careful implementation to avoid pitfalls.

C Linked Lists

A linked list is a data structure consisting of nodes that hold data and pointers to the next node.

Node Structure

struct Node {
    int data;
    struct Node *next; // Pointer to the next node
};

Creating a Linked List

#include <stdio.h>
#include <stdlib.h>

struct Node {
    int data;
    struct Node *next;
};

int main() {
    struct Node *head = (struct Node *)malloc(sizeof(struct Node));
    head->data = 1;
    head->next = NULL; // Initialize the first node
    
    struct Node *second = (struct Node *)malloc(sizeof(struct Node));
    second->data = 2;
    second->next = NULL;
    
    head->next = second; // Link first node to second
    
    printf("First node data: %d\n", head->data);
    printf("Second node data: %d\n", head->next->data);
    
    free(head);
    free(second);
    
    return 0;
}

Traversing a Linked List

#include <stdio.h>
#include <stdlib.h>

struct Node {
    int data;
    struct Node *next;
};

void printList(struct Node *node) {
    while (node != NULL) {
        printf("%d -> ", node->data);
        node = node->next;
    }
    printf("NULL\n");
}

int main() {
    struct Node *head = (struct Node *)malloc(sizeof(struct Node));
    head->data = 1;
    head->next = NULL;
    
    struct Node *second = (struct Node *)malloc(sizeof(struct Node));
    second->data = 2;
    second->next = NULL;
    
    head->next = second;
    
    printList(head);
    
    free(head);
    free(second);
    
    return 0;
}

Conclusion

Linked lists are a flexible data structure that allows dynamic memory allocation and efficient insertions/deletions.

C Bitwise Operations

Bitwise operations are used to manipulate individual bits of data, providing efficient low-level programming capabilities.

Types of Bitwise Operators

  • AND (&): Sets each bit to 1 if both bits are 1.
  • OR (|): Sets each bit to 1 if at least one of the bits is 1.
  • XOR (^): Sets each bit to 1 if only one of the bits is 1.
  • NOT (~): Inverts all the bits.
  • Left Shift (<<): Shifts bits to the left, filling with 0s.
  • Right Shift (>>): Shifts bits to the right.

Example Program

#include <stdio.h>

int main() {
    int a = 5;  // 0101 in binary
    int b = 3;  // 0011 in binary
    
    printf("a & b: %d\n", a & b); // AND
    printf("a | b: %d\n", a | b); // OR
    printf("a ^ b: %d\n", a ^ b); // XOR
    printf("~a: %d\n", ~a); // NOT
    printf("a << 1: %d\n", a << 1); // Left Shift
    printf("a >> 1: %d\n", a >> 1); // Right Shift
    
    return 0;
}

Conclusion

Bitwise operations are essential for low-level programming and optimization in C.
