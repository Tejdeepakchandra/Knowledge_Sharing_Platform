What is PHP?

PHP is a server-side scripting language designed primarily for web development but also used as a general-purpose programming language.

History of PHP

PHP was created by Rasmus Lerdorf in 1994 and has evolved significantly over the years.

Key Features of PHP

  • Open Source: PHP is free to use and distribute.
  • Cross-Platform: Runs on various platforms like Windows, Linux, and macOS.
  • Database Support: Easily connects to various databases like MySQL, PostgreSQL, and SQLite.
  • Rich Library: Extensive built-in functions and libraries.

Conclusion

PHP is a powerful tool for making dynamic and interactive web pages. Its ease of use and flexibility make it a popular choice among developers.

PHP Installation & Setup

Follow these steps to install PHP on different operating systems and set up your development environment.

Installing PHP on Windows

  1. Download PHP from the official website php.net
  2. Extract the files to a directory (e.g., C:\php)
  3. Add PHP to your system PATH:
    • Right-click on "This PC" → Properties → Advanced system settings
    • Click "Environment Variables" → Under "System variables", find "Path" and click "Edit"
    • Add the path to PHP directory (e.g., C:\php)
  4. Verify installation by opening Command Prompt and typing: 
    php -v

Installing PHP on Linux

  1. Open terminal and install PHP: 
    sudo apt update
sudo apt install php
  2. Verify installation: 
    php -v

Installing PHP on macOS

  1. Install Homebrew if you haven't already: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install PHP: 
    brew install php
  3. Verify installation: 
    php -v

Writing Your First PHP Script

Create a file named hello.php with the following content:

<?php
echo "Hello, World!";
?>

Run the script from the command line:

php hello.php

Conclusion

After following these steps, you'll have a working PHP development environment ready to create and run PHP scripts.

PHP Syntax Basics

PHP is a server-side scripting language with a straightforward syntax.

Structure of a PHP Script

A basic PHP script consists of PHP tags and statements:

<?php
// PHP code goes here
echo "Hello, PHP!";
?>

Key Components

  • PHP Tags: PHP code is enclosed within <?php and ?> tags.
  • Statements: Instructions that end with a semicolon (;)
  • Comments: Explanatory notes ignored by the interpreter (// for single-line, /* */ for multi-line)

Basic Syntax Rules

  • PHP is case-sensitive
  • Statements end with a semicolon (;)
  • Whitespace is ignored (except in strings)
  • Variables start with a dollar sign ($)

Conclusion

Understanding the basic syntax is essential for writing PHP scripts effectively.

PHP Variables & Data Types

Variables in PHP are containers for storing data values, each with a specific type.

Declaring Variables

Variable declaration in PHP is straightforward:

$age = 25;          // Integer
$price = 19.99;     // Float
$grade = 'A';       // String
$isStudent = true;  // Boolean

Basic Data Types

  • int: Integer numbers
  • float: Floating point numbers
  • string: Sequence of characters
  • bool: Boolean values (true/false)
  • array: Collection of values
  • object: Instance of a class
  • NULL: Represents a variable with no value

Variable Naming Rules

  • Can contain letters, digits, and underscores
  • Must begin with a letter or underscore
  • Cannot be a PHP keyword
  • Case-sensitive

Conclusion

Understanding variables and data types is fundamental to PHP programming, as they define how data is stored and manipulated.

PHP Arrays

Arrays in PHP are collections of values stored in a single variable.

Declaring Arrays

Syntax for array declaration:

$numbers = array(1, 2, 3, 4, 5); // Indexed array
$fruits = ["Apple", "Banana", "Cherry"]; // Short array syntax
$person = ["name" => "John", "age" => 30]; // Associative array

Accessing Array Elements

Array elements are accessed using their index:

echo $fruits[0]; // Outputs: Apple
echo $person["name"]; // Outputs: John

Multi-dimensional Arrays

PHP supports arrays of arrays (2D, 3D, etc.):

$matrix = [
    [1, 2, 3],
    [4, 5, 6]
];

echo $matrix[1][2]; // Outputs: 6

Conclusion

Arrays are fundamental for storing and processing collections of data in PHP.

PHP Strings

In PHP, strings are sequences of characters.

String Declaration

Multiple ways to declare strings:

$str1 = "Hello"; // Double quotes
$str2 = 'World'; // Single quotes

String Functions

PHP provides many built-in string functions:

echo strlen($str1); // String length
echo strtoupper($str1); // Convert to uppercase
echo strtolower($str2); // Convert to lowercase

Common String Functions

  • strlen(): Returns length of string
  • strtoupper(): Converts to uppercase
  • strtolower(): Converts to lowercase
  • strpos(): Finds position of substring

Conclusion

Strings are essential for text processing in PHP, and understanding string functions is crucial for effective programming.

PHP File Handling

File handling in PHP allows you to read from and write to files, enabling data persistence.

File Operations

  • Closing a File: Use fclose() to close a file.

Example Program
