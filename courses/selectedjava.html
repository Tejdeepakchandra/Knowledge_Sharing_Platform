What is Java?

Java is a class-based, object-oriented programming language designed to have minimal implementation dependencies. It allows developers to "Write Once, Run Anywhere" (WORA), meaning Java code can run on any platform that supports Java without recompilation.

History of Java

Java was developed by James Gosling at Sun Microsystems in May 1995. Later, Oracle Corporation acquired Java, making it one of the most widely used languages for desktop, web, and mobile applications.

  • Originally named Oak, later renamed Java after Java coffee.
  • First publicly released in 1996 as Java 1.0.
  • In 2006, Sun Microsystems released a large portion of the Java Virtual Machine (JVM) as open-source software.
  • By 2007, the core JVM code became fully available under open-source distribution.

Key Features of Java

  • Platform Independent: Java bytecode runs on any OS with a JVM.
  • Object-Oriented: Java follows the OOP principles of Abstraction, Encapsulation, Inheritance, and Polymorphism.
  • Simplicity: Java eliminates complex features like pointers and multiple inheritance.
  • Robustness: Java has strong memory management, garbage collection, and exception handling.
  • Security: Java runs in a secure environment, preventing unauthorized access and vulnerabilities.
  • Multithreading: Java supports concurrent execution of multiple tasks for high-performance applications.
  • Portability: Java code can be executed on any machine, thanks to its architecture-neutral bytecode.
  • High Performance: Uses Just-In-Time (JIT) compiler to optimize code execution.

How Java Code Executes?

  1. Writing the Program: Developers write Java code in a text editor or IDE.
  2. Compiling the Code: The Java compiler (javac) converts source code into bytecode.
  3. Running the Program: The Java Virtual Machine (JVM) interprets and executes the bytecode.

Conclusion

Java is a powerful and versatile programming language used for a wide range of applications. Its platform independence, strong security, and OOP principles make it one of the most popular programming languages in the world.

Java Installation & Setup

Follow these steps to install Java on different operating systems and set up environment variables.

Setting Up Environment Variables

Setting up environment variables helps the system locate Java tools needed to run Java programs. Ensure that the Java Development Kit (JDK) is installed and its directory is known.

  • JAVA_HOME: Points to the JDK installation directory.
  • PATH: Specifies where the OS should look for executable files.

Setting Environment Variables in Windows

  1. Download and install the latest JDK from the Oracle website.
  2. Press Windows + S, search for “Environment Variables,” and select Edit the system environment variables.
  3. In the System Properties window, click Environment Variables.
  4. Under System Variables, click New and set:
    • Variable name: JAVA_HOME
    • Variable value: Path to the JDK installation folder (e.g., C:\Program Files\Java\jdk-23)
  5. Update the PATH variable:
    • Locate the bin folder inside JDK (e.g., C:\Program Files\Java\jdk-23\bin).
    • Under System Variables, select Path → Edit → New.
    • Add the copied path and click OK.
  6. Verify installation using Command Prompt:
    • java --version
    • javac --version

Setting Environment Variables in Linux

  1. After installing JDK, open the terminal and run: 
    export JAVA_HOME=/path/to/your/jdk
  2. Update the PATH variable: 
    export PATH=$PATH:$JAVA_HOME/bin
  3. Make the changes permanent by adding these lines to your shell configuration file:
    • Bash: 
      nano ~/.bashrc
    • Zsh: 
      nano ~/.zshrc
    • Fish: 
      nano ~/.config/fish/config.fish
    Add the export commands and save the file. Then apply changes: 
    source ~/.bashrc
  4. Verify installation: 
    echo $JAVA_HOME
    echo $PATH
    java --version
    javac --version

Setting Environment Variables in macOS

  1. Open the terminal and edit the shell profile:
    • Bash (Older macOS): 
      nano ~/.bash_profile
    • Zsh (Default for macOS Catalina and later): 
      nano ~/.zshrc
  2. Add the following lines at the end of the file: 
    export JAVA_HOME=$(/usr/libexec/java_home)
        export PATH=$JAVA_HOME/bin:$PATH
  3. Apply the changes: 
    source ~/.zshrc
  4. Verify installation: 
    echo $JAVA_HOME
    echo $PATH
    java --version
    javac --version

Conclusion

After following these steps, your Java environment variables will be correctly set, allowing smooth execution of Java programs across different operating systems.

Java Syntax

Java is an object-oriented programming language known for its simplicity, portability, and robustness. Its syntax is closely aligned with C and C++, making it easier to understand.

Structure of a Java Program

A basic Java program consists of several components that create a functional application. Below is a simple "Hello World" program:

                // FileName : "Radha.java".
                public class Radha {
                    public static void main(String args[]) {
                        System.out.println("Hello World");
                    }
                }

Key Components of a Java Program

  • Class Declaration: Every Java program starts with a class using the class keyword.
  • Object: An instance of a class that represents an entity with behavior and state.
  • Main Method: The entry point of the Java program.
  • Statements: Each line inside a method ends with a semicolon (;).

Java Syntax Key Elements

  • Comments: Java supports single-line, multi-line, and documentation comments.
  • Source File Naming Rules: Public class names must match the file name.
  • Case Sensitivity: Java is case-sensitive.
  • Naming Conventions: Class names start with uppercase, methods with lowercase.

Compiling and Running a Java Program

                javac Radha.java
                java Radha

After following these rules, you will have a well-structured Java program with correct syntax.

Java Variables & Data Types

Variables in Java store data values and are associated with data types that determine the type of data they can hold.

Declaring Variables

A variable in Java is declared using a specific data type followed by the variable name. For example:

                int age = 25;
                double price = 99.99;
                char grade = 'A';
                boolean isJavaFun = true;

Types of Variables

  • Local Variables: Declared inside a method or block and accessible only within that scope.
  • Instance Variables: Belong to an object and are declared outside methods but inside a class.
  • Static Variables: Declared with the static keyword and shared among all objects of a class.

Primitive Data Types

Java has eight primitive data types:

  • byte: 8-bit integer (-128 to 127)
  • short: 16-bit integer (-32,768 to 32,767)
  • int: 32-bit integer (-2^31 to 2^31-1)
  • long: 64-bit integer (-2^63 to 2^63-1)
  • float: 32-bit floating point
  • double: 64-bit floating point
  • char: 16-bit Unicode character
  • boolean: Represents true or false

Example Program

Below is an example demonstrating variables and data types in Java:

                // FileName : "DataTypesExample.java"
                public class DataTypesExample {
                    public static void main(String[] args) {
                        int num = 100;
                        double pi = 3.14159;
                        char letter = 'J';
                        boolean isCodingFun = true;
                        
                        System.out.println("Integer: " + num);
                        System.out.println("Double: " + pi);
                        System.out.println("Character: " + letter);
                        System.out.println("Boolean: " + isCodingFun);
                    }
                }

Conclusion

Understanding variables and data types is crucial in Java, as they define how data is stored and manipulated in a program.

Java Operators

Operators in Java are special symbols that perform operations on variables and values. They are essential for manipulating data in programs.

Types of Operators

  • Arithmetic Operators: Used for mathematical calculations (e.g., +, -, *, /, %).
  • Relational Operators: Compare two values (e.g., ==, !=, >, <, >=, <=).
  • Logical Operators: Combine boolean expressions (e.g., &&, ||, !).
  • Assignment Operators: Assign values to variables (e.g., =, +=, -=, *=).
  • Bitwise Operators: Operate on bits (e.g., &, |, ^, ~, <<, >>).
  • Unary Operators: Operate on a single operand (e.g., ++, --).

Example Program

Here's an example showcasing some common operators:

        // FileName: "OperatorsExample.java"
        public class OperatorsExample {
            public static void main(String[] args) {
                int a = 10, b = 5;
                // Arithmetic
                System.out.println("Sum: " + (a + b));
                System.out.println("Modulus: " + (a % b));
                // Relational
                System.out.println("Is a > b? " + (a > b));
                // Logical
                boolean x = true, y = false;
                System.out.println("x && y: " + (x && y));
                // Assignment
                a += 3; // a = a + 3
                System.out.println("Updated a: " + a);
                // Unary
                a++;
                System.out.println("Incremented a: " + a);
            }
        }

Conclusion

Operators are the building blocks of computation in Java, enabling you to perform everything from simple math to complex logical evaluations.

Java Control Statements

Control statements in Java dictate the flow of execution in a program, allowing decisions and repetitions based on conditions.

Types of Control Statements

  • Decision-Making: if, if-else, switch.
  • Looping: for, while, do-while.
  • Branching: break, continue, return.

Example Program

Here's an example using if-else and a for loop:

        // FileName: "ControlExample.java"
        public class ControlExample {
            public static void main(String[] args) {
                int number = 10;
                // Decision-Making
                if (number > 0) {
                    System.out.println(number + " is positive");
                } else {
                    System.out.println(number + " is negative or zero");
                }
                // Looping
                System.out.println("Counting from 1 to 5:");
                for (int i = 1; i <= 5; i++) {
                    System.out.println(i);
                }
            }
        }

Conclusion

Control statements give you the power to make your Java programs dynamic, adapting to different inputs and conditions.

Java Arrays

Arrays in Java are used to store multiple values of the same type in a single variable.

Declaring and Using Arrays

  • Syntax: type[] arrayName = new type[size];
  • Access: Use index starting from 0.

Example Program

        // FileName: "ArraysExample.java"
        public class ArraysExample {
            public static void main(String[] args) {
                int[] numbers = {1, 2, 3, 4, 5};
                System.out.println("First element: " + numbers[0]);
                System.out.println("Array length: " + numbers.length);
                System.out.println("All elements:");
                for (int i = 0; i < numbers.length; i++) {
                    System.out.println(numbers[i]);
                }
            }
        }

Conclusion

Arrays provide an efficient way to manage collections of data in Java.

Java Strings

Strings in Java are objects that represent sequences of characters, handled by the String class.

Key Features

OOP Basics in Java

Object-Oriented Programming (OOP) in Java revolves around objects and classes, promoting modularity and reusability.

Core Principles

  • Encapsulation: Hiding data with access modifiers.
  • Inheritance: Reusing code via parent-child relationships.
  • Polymorphism: Multiple forms of a method.
  • Abstraction: Hiding complexity with interfaces/abstract classes.

Conclusion

OOP principles form the backbone of Java’s design philosophy.

Java Inheritance

Inheritance allows a class to inherit properties and methods from another class using the extends keyword.

Example Program

        // FileName: "InheritanceExample.java"
        class Animal {
            void eat() { System.out.println("Eating..."); }
        }
        class Dog extends Animal {
            void bark() { System.out.println("Barking..."); }
        }
        public class InheritanceExample {
            public static void main(String[] args) {
                Dog dog = new Dog();
                dog.eat();
                dog.bark();
            }
        }

Conclusion

Inheritance promotes code reuse and establishes a hierarchy between classes.

Java Polymorphism

Polymorphism allows methods to take multiple forms, achieved through method overriding or overloading.

Example Program

        // FileName: "PolymorphismExample.java"
        class Animal {
            void sound() { System.out.println("Some sound..."); }
        }
        class Cat extends Animal {
            void sound() { System.out.println("Meow"); }
        }
        public class PolymorphismExample {
            public static void main(String[] args) {
                Animal myCat = new Cat();
                myCat.sound();
            }
        }

Conclusion

Polymorphism enhances flexibility in Java programs.

Java Abstraction & Interfaces

Abstraction hides implementation details, achieved using abstract classes or interfaces.

Example Program

        // FileName: "AbstractionExample.java"

            void draw();

            public void draw() { System.out.println("Drawing Circle"); }

Conclusion

Abstraction simplifies complex systems by exposing only necessary details.

Java JDBC

JDBC (Java Database Connectivity) allows Java programs to interact with databases.

Example Program

        // FileName: "JDBCExample.java"
        import java.sql.*;
        public class JDBCExample {
            public static void main(String[] args) {
                try {
                    Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/test", "root", "password");
                    Statement stmt = conn.createStatement();
                    ResultSet rs = stmt.executeQuery("SELECT * FROM users");
                    while (rs.next()) {
                        System.out.println(rs.getString("name"));
                    }
                    conn.close();
                } catch (SQLException e) {
                    System.out.println("Error: " + e.getMessage());
                }
            }
        }

Conclusion

JDBC connects Java applications to databases for dynamic data handling.
